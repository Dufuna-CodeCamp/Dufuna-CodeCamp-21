IPhone X

Price: $1000


Description

The iPhone X was Apple's tenth-anniversary iPhone, released in November 2017. It was a major redesign from previous iPhones, with an all-glass front and back, a 5.8-inch OLED display, and no home button. Instead of Touch ID, the iPhone X used Face ID, a facial recognition system that allowed users to unlock their phones and authorize payments with a glance.

Features

  • 5.8-inch Super Retina OLED display
  • A11 Bionic chip with Neural Engine
  • Face ID facial recognition system
  • Wireless charging
  • Animoji feature
  • TrueDepth camera system for Portrait Mode selfies
